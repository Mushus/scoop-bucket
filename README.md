# scoop-bucket
scoop bucket

## How to Use

### Install scoop

See [lukesampson/scoop](https://github.com/lukesampson/scoop).

### Add bucket

```
scoop bucket add mushus git@github.com:Mushus/scoop-bucket.git
```

### Install apps

```
scoop update
scoop install [app]
```

VST Path:
```
%USERPLOFILE%\.vst
```
